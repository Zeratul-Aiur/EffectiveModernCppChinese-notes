## 条款七：区别使用 () 和 {} 创建对象

- 使用花括号，创建并指定一个容器的初始元素变得很容易。
- 花括号初始化可以防止变窄变化。无参数初始化解析成函数声明。

## 条款八：优先考虑 nullptr 而非 0 和 NULL

- `nullptr`是一个类型为`std::nullptr_t`的字面值常量，可以隐式转换为任何指针类型。
